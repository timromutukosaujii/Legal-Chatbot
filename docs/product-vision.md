# Product Vision Justification

The project evolves from a single chatbot into an **AI Legal Assistant Platform** with trust and control features.

## Academic Rationale

User authentication and subscription features were included to simulate real-world deployment conditions and evaluate system usability under controlled access environments.

## Why This Matters

- Authentication enables per-user auditability and measurable usage patterns.
- Tiered plans enable controlled load and fairness testing (free-tier limits versus pro-tier access).
- Transparency features (confidence, citations, reasoning view) improve explainability and trust calibration.
- Saved conversations improve longitudinal usability analysis in user studies.
